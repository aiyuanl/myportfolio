2019 Med AI Jam比赛

  • by:Aiyuan Li，*
  • date:2019.02.12-2019.02.22

比赛简介：
来自医学院的2名学生和计算机学院的五名学生一起，在两周内设计并制作出最终可运行的医学AI app

项目简介：
通过每个时间段输入的病人的各项数值（心脏收缩压SBP， 舒张压DBP，心率HR，呼吸是否规律）来预测是否提醒病人做好预防库欣反射的准备

主要职责/视频简介：
在向医学生询问了客户想法后，确定了目标，自学并掌握了Android Studio，拟定了UI初稿，之后完成了UI的全部工作，帮助完成前端工作，绘制了图表，以及构建bottom navigation bar，并帮助其他组员完成前端和后端的融合。

获得奖项：“Most Innovative use of AI"

相关技能展示：html, css, Android Studio, Java
