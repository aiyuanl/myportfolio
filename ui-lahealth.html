  • by:Aiyuan Li，*

项目简介：
使用Kaggle提供的健康类数据库设计并完成一个网页。经协商，小组决定分析洛杉矶地区餐厅的卫生检测结果，实现数据可视化

主要职责：
作为沟通组长CCO，督促其余3名组员按时完成项目进度，编写html完成各种chart的制作，完善UI

过程记录：
完成千字项目建议书，分析产品对比，项目的设计方向策划
课堂10分钟策划演讲，将未来的项目进程报告给教授
作为网页制作人员，使用队友给出的json文件在html文件中描绘图表，完成剩余的网页设计
完成千字项目总结报告，包括过程总结，经验教训总结，未来构想
10分钟完结演讲报告+5分钟Q&A

相关技能展示：Visual Studio Code, 人际交流管理，项目写作，html, CSS, javascript
